## üöÄ Current Project: Bird Class Identifier(Flutter App)

<p align="justify">
This application is designed to identify and classify different <strong>Bird Species</strong> using images. The main goal of the app is to help users easily recognize birds through a simple and user-friendly mobile interface built with Flutter.
<br/><br/>
The app can identify ten classes of birds, which include Crow, Eagle, Hummingbird, Owl, Parrot, Peacock, Penguin, Pigeon, Sparrow, and Swan. When a user selects or uploads an image of a bird, the application analyzes it and displays the bird‚Äôs class along with basic information.
</p>
